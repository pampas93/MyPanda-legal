# MyPanda Legal Pages

This folder is an independent git repository intended for publishing legal pages.

## Files
- `terms.md`
- `privacy.md`

## GitHub Pages (quick setup)
1. In the `MyPanda-legal` repo settings, enable GitHub Pages.
2. Set source to `Deploy from a branch`, branch `main`, folder `/ (root)`.
3. GitHub will provide a public URL you can use in-app.
